I included the folder "Data" with my source code. Please keep it in the same folder. It contains"
a) Training data 
b) Test data
c) Stop words (stopWords.txt)

Compile:
---------
javac Main.java

Run:
-----
The program can be run in two modes:

1) Mode 1 (non-parametric):
java Main

In this mode, program use default values for various parameters.
Default values are:
iteration number = 100
learning rate = 0.01
regularization parameter = 10
initial weight value = 0.0
[prior strength/smoothing = 2.0] optional for extra credit
[minimum words = 5.0] optional for extra credit

2) Mode 2 (parametric):
java Main <iteration number> <learning rate> <regularization parameter value (lamda)> <initial weight value> [prior strength] [minWords]

parameters [prior strength] are  [minWords] are prior strength (instead of laplace smoothing we can use k>1) and we can eliminate less frequent word like 5. These parameters are  optional for extra credit. 
For example:
java Main 1000 0.01 1 0
java Main 1000 0.01 1 0 2
java Main 1000 0.01 1 0 2 5
